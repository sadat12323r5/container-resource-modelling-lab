# Thesis B Slides

Open `thesis_b_slides.html` in a browser.

Controls:

- Right arrow, Page Down, Space, or click: next slide
- Left arrow, Page Up, or Backspace: previous slide
- Home / End: first / last slide

To export to PDF, open the deck in a browser and print to PDF using landscape
orientation. The deck is built from local files only; figures are loaded from
`../../results/figures/`.
