# Thesis B Slides

Open either HTML deck in a browser, or use an exported PDF when available:

- `thesis_b_slides.html` - editable browser deck
- `thesis_b_slides.pdf` - exported PDF version
- `service_time_fitting_slides.html` - focused editable deck for the M0/M1/M2/M3 fitting, CDF validation, histograms, ANOVA, and interpretation story

Controls:

- Right arrow, Page Down, Space, or click: next slide
- Left arrow, Page Up, or Backspace: previous slide
- Home / End: first / last slide
- `E`: toggle edit mode
- In edit mode, click slide text to edit it directly
- `Ctrl+S`: save edits in this browser

The deck has an editing toolbar in the top-right corner:

- `Edit`: toggle live text editing
- `Add Text`: add a movable text box to the current slide
- `Delete`: delete the selected movable item
- `Save`: save edits to browser local storage
- `Download HTML`: download an edited standalone HTML copy
- `Reset`: clear saved browser edits and reload the original file

Layout editing:

- Click a card, figure, equation, callout, or added text box to select it
- Hold `Shift` and drag the selected item to move it around the slide
- Added text boxes can be edited like normal slide text
- Save after moving/adding/deleting if you want the browser to remember the layout

To regenerate the PDF, open the deck in a browser and print to PDF using
landscape orientation. The deck is built from local files only; figures are
loaded from `../../results/figures/`.
